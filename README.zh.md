# OneMind

[English](./README.md)

在 Obsidian 中将 Markdown 笔记直接作为交互式思维导图编辑。OneMind 会把标题和无序列表渲染为清爽的 SVG + DOM 思维导图，并把所有编辑实时写回纯 Markdown。

## 功能特性

- **Markdown 原生**：标题和项目符号列表直接映射为思维导图树，编辑结果仍然保存为普通 Markdown。
- **键盘优先**：`Tab` 新增子节点，`Enter` 新增同级节点，`F2` 编辑，方向键导航，`Alt + 方向键` 调整结构。
- **拖拽编辑**：支持节点拖拽排序、换父节点，触屏设备支持长按菜单。
- **两种布局**：支持经典向右树形布局和双侧平衡布局。
- **Emoji 与标签**：给节点添加 Emoji 图标和 `#标签`，并按标签过滤导图。
- **跨分支关联线**：自动为 `[[#标题]]` 和 `[[标题]]` 这类 wikilink 绘制虚线关联。
- **大纲面板**：内置可折叠大纲，方便浏览大型导图。
- **撤销 / 重做**：支持 `Ctrl/Cmd + Z` 和 `Ctrl/Cmd + Shift + Z`。
- **导出**：支持将整张导图或选中分支导出为 SVG 或 PNG。
- **多选操作**：支持 `Ctrl/Cmd + 点击`、`Shift + 点击` 和 `Ctrl/Cmd + A`。
- **复制 / 粘贴**：支持把分支复制为 Markdown，也可以把 Markdown 粘贴成新节点。
- **节点链接**：可复制任意节点的 `obsidian://open` 链接。
- **中英文界面**：内置中文与英文 UI。
- **移动端友好**：响应式工具栏与触控支持。

## 快捷键

| 快捷键 | 作用 |
|---|---|
| `Tab` | 新增子节点 |
| `Enter` | 新增同级节点 |
| `F2` | 编辑当前节点 |
| `Delete` / `Backspace` | 删除当前节点 |
| `Space` | 折叠 / 展开 |
| `/` | 聚焦搜索 |
| `Alt/Option + 方向键` | 上移 / 下移 / 缩进 / 取消缩进 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `Ctrl/Cmd + A` | 全选节点 |
| `Ctrl/Cmd + =` / `-` | 放大 / 缩小 |
| `Ctrl/Cmd + Shift + F` | 导图适应窗口 |
| `Ctrl/Cmd + F` | 聚焦当前选中节点 |

## 安装

1. 从 [最新发布页](https://github.com/qiuos/onemind/releases) 下载 `main.js`、`manifest.json` 和 `styles.css`。
2. 在你的仓库目录下创建 `.obsidian/plugins/onemind/` 文件夹。
3. 将这三个文件放入该文件夹。
4. 打开 Obsidian：设置 > 第三方插件，刷新并启用 **OneMind**。

`styles.css` 为可选文件，但推荐安装。

## 使用方法

1. 打开任意包含标题或无序列表的 Markdown 笔记。
2. 点击左侧边栏脑图图标，或在命令面板执行 **“以思维导图打开当前笔记”**。
3. 直接编辑节点、调整结构、导出图片；所有修改都会自动写回 Markdown。

## 设置项

| 设置项 | 默认值 | 说明 |
|---|---|---|
| 语言 | 中文 | 支持中文 / 英文界面 |
| 自动保存延迟 | 300 ms | 编辑后回写 Markdown 的延迟 |
| 默认展开深度 | 99 | 打开时默认展开层级，99 表示全部展开 |
| 布局方向 | 向右 | 向右树形 / 双侧平衡 |
| 布局动画 | 开启 | 编辑后节点移动动画 |
| 显示关联线 | 开启 | 为本地 wikilink 绘制虚线关联 |
| 分支颜色 | 5 个预设色 | 一级分支颜色配置 |
| PNG 导出倍率 | 2x | PNG 分辨率倍率 |
| 透明 PNG 背景 | 关闭 | 导出 PNG 时是否保留透明背景 |
